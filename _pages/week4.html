    • Monday, June 3rd - Autobiographical Memory (#13/28 of class):

    Learning Objectives:
    1. LO1: Continue to build a classroom culture & discuss science communication
    2. LO2: Apply principles of good science communication
    3. LO3: Define, identify, and apply previous constructs that we have discussed in class
    4. LO4: Describe the basic foundamental principles of autobiographical memory
    5. LO5: Summarize and critically analyze academic journal articles

    Agenda:

    1. Quiz on material covered since last class: multiple choice & short answer (LO3).
    2. Demo on fictional first memories; what does several deficient autobiographical memory look like (Palombo et al., 2015)? (LO4)

    7. Reflections on class and any concerns/questions you, students, have.

    Learning Objectives:
    1. LO1: Continue to build a classroom culture & discuss science communication.
    2. LO2: Define, identify, and apply previous constructs that we have discussed in class
    3. LO3: Describe the basic principles thought to underlie motivating reasoning, cognitive disonance, confirmation bias.
    4. LO4: Deconstruct and analyze the findings of an academic journal article.
    5. LO5: Apply these motivated reasoning findings to science communication principles.

    Agenda:

    1. Quiz on material covered since the last quiz: multiple choice & short answer (LO2).
